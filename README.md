# مشروع الموقع

## نظرة عامة

هذا المشروع هو موقع ويب يحتوي على ملفات متعددة مثل ملفات CSS و JavaScript وصور وملفات HTML. يتم استخدام Python لإدارة التطبيق.

## هيكل المشروع

- `app.py`: الملف الرئيسي لتشغيل التطبيق.
- `comtypes_patch.py`: ملف يحتوي على تعديلات خاصة.
- `requirements.txt`: يحتوي على قائمة المكتبات المطلوبة.
- `downloads/`: يحتوي على الملفات التي يتم تنزيلها.
- `instance/`: يحتوي على قاعدة البيانات `site.db`.
- `static/`: يحتوي على الملفات الثابتة مثل CSS و JavaScript والصور.
  - `css/style.css`: ملف أنماط CSS.
  - `img/logo.png`: صورة الشعار.
  - `js/script.js`: ملف JavaScript.
- `templates/`: يحتوي على ملفات HTML.
  - `index.html`: الصفحة الرئيسية.
  - `main.html`: صفحة إضافية.
- `uploads/`: يحتوي على الملفات التي يتم رفعها.

## كيفية التشغيل

1. تأكد من تثبيت Python على جهازك.
2. قم بتثبيت المكتبات المطلوبة باستخدام:

   ```bash
   pip install -r requirements.txt
   ```

3. قم بتشغيل التطبيق باستخدام:

   ```bash
   python app.py
   ```

## ملاحظات إضافية

- تأكد من إعداد قاعدة البيانات `site.db` بشكل صحيح.

- يمكن تعديل الملفات الثابتة والصفحات حسب الحاجة.
